# hand386-image
Hand386 Image Creator

## Description
Will create a bootable compact flash image for the Hand386 system.

## Features
- Create 1GB disk image
- Format as bootable for DOS 6.22

## Usage
Ensure you have Homebrew installed.
It will install QEMU.
